# Muday Marketing Solutions - Website

A complete TypeScript website for Muday Marketing Solutions with all functionality, animations, and interactive elements.

## Features

- ✅ Fully responsive design
- ✅ Smooth animations using Framer Motion
- ✅ Interactive pricing tabs (SEO/Design/Content)
- ✅ Mobile-friendly navigation with dropdown menus
- ✅ Contact form with validation
- ✅ Scroll-triggered animations
- ✅ Modern UI with Tailwind CSS
- ✅ TypeScript for type safety

## Tech Stack

- **React 18** - UI library
- **TypeScript** - Type safety
- **Vite** - Build tool
- **Tailwind CSS** - Styling
- **Framer Motion** - Animations
- **Lucide React** - Icons

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

The site will be available at `http://localhost:5173`

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Sections

1. **Hero** - Main landing section with CTA buttons
2. **About** - Company introduction with statistics
3. **Services** - Four main service offerings
4. **Pricing** - Three pricing tiers with tab switching
5. **Industries** - Industry expertise showcase
6. **Process** - Three-step workflow
7. **Portfolio** - Case studies and success stories
8. **Blog** - Latest blog posts
9. **Contact** - Contact form and information
10. **Footer** - Links and newsletter signup
